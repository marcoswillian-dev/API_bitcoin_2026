# 📈 API Bitcoin 2026

Projeto desenvolvido em Python para consumir dados históricos do Bitcoin através da API da CoinGecko, realizar tratamento dos dados e gerar análises utilizando Pandas e gráficos com Matplotlib.

---

## 🚀 Objetivo do Projeto

O objetivo deste projeto é praticar conceitos de:

- Consumo de APIs REST;
- Manipulação de dados em Python;
- Tratamento de JSON;
- Criação de DataFrames;
- Análise de séries temporais;
- Visualização de dados.

---

## 🛠️ Tecnologias utilizadas

- Python 3
- Requests
- Pandas
- Matplotlib
- CoinGecko API
- Git e GitHub
